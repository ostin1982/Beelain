# Сайт для beelain
***
### О проекте:
Проект создан для изучения HTML, CSS и JS.
***
### Что реализованно:
* Резиновая вёрстка сайта (**desktop first**)
* Select menu для выбора города и языка
* Dropdown menu
* Search bar
* Phone Mask
* Visible Password
* Validate
***
### Используемые технологии:
* HTML (**БЭМ Nested**)
* SCSS (**Flexbox** и **Grid Layout**)  
* JavaScript
* Webpack
